The SQL Saturday Foundation

The SQL Saturday Foundation will be established by Redgate Software as an independent, community run non-profit foundation to support the SQL Saturday franchise. While Redgate maintains a permanent board seat on the foundation, the foundation is steered and managed by a group of unpaid, community volunteers.

This foundation will be established in 2021.

You can learn more at the SQL Saturday Foundation website (coming soon).
